# parking-control-api

### API para controle de vagas de estacionamento de condomínios utilizando Spring Boot

**Projetos utilizados:**

- Spring MVC
- Spring Data JPA
- Spring Validation

**Métodos:**

- getAll
- getOne
- post
- delete
- update
- Global Custom Date


Obs: Fazer conexão com o banco de dados em "src/main/resources/application.properties".
